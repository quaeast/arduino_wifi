# arduino_wifi

## 芯片和配件

* Arduino UNO
* Esp 8266-01s
* MG995

## 连接

### Arduino -- Esp 8266-01s

| Esp-01s | Arduino      |
| ------- | ------------ |
| 3v3     | 3v3          |
| GND     | GND          |
| Tx      | 6（soft Rx） |
| Rx      | 7（soft Tx） |

### Adruino -- MG996

| MG996  | Arduino |
| ------ | ------- |
| Red    | 5v      |
| Black  | GND     |
| Yellow | 9       |
